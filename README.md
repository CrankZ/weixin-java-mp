# 主要功能

1.菜单设置<br>
2.接受消息与自动回复，目前是自动回复相同的内容<br>
3.菜单单击事件，不同的按钮有不同的事件，包括文字、图文回复和弹出网页<br>
4.内置咨询网页<br>
5.内置邮件发送服务，咨询页面的内容会主动发送到相关负责人的邮箱<br>
